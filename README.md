### About

Developed a online bookstore application enabling user registration, book browsing, selection, secure transactions, and payment receipts. Admins can manage books, track sales, and make necessary updates utilizing MySQL, Java, Spring boot, html, Css, JDBC, Servlets, Bootstrap



<span style="color:blue">**Admin Access :-**</span>
- Add New Books.
- View Books Available.
- Remove Books.
- Increase Books Amount.

<span style="color:blue">**Users Access:-**</span>
- Create New Account or Register.
- Login.
- View Available Books.
- Select Books to Buy.
- Select Books Quantity.
- Buy Books.
- Get Payment Receipt.

### Technologies used:-
1. Front-End Development:
- HTML
- CSS
- Javascript
- BootStrap

2. Back-End Development:
- Java [JDK 8+]
- JDBC
- Servlet

3. Database:
- MySql


Note:- This is a Sample Project, we have not much considered of web security.
